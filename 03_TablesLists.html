HTML List

Ungeordnete Listr ("ul")

Es gibt verschiedene Aufzählungszeichen, welche man auf eine Liste anwenden kann:

Aufzählungszeichenlhlungszeichen:

  • disc "list-style-type:disc"
  • Circle "list-style-type:circle"
  • Square "list-style-type:square"
  • None "list-style-type:none"

Geordnete Liste ("ol")

    Überschrift
  • type = "1" Nummeriert
  • type = "A" Großes Alphabet
  • type = "a" kleines Alphabet
  • type = "I" große römische Ziffern
  • type = "i" kleine römische Ziffern

HTML List Tags

  • < ul > Deffiniert eine ungeordnete Liste
  • < ol > Deffiniert eine geordnete Liste
  • < li > Deffiniert ein Item in der Liste
  • < dl > Deffiniert eine Description List
  • < dt > Deffiniert einen Term in der Description List
  • < dd > Deffiniert eine Beschreibung unter einem Term

CSS List Aufzählungszeichen

Mithilfe des Kommandos list-style-type kann in css und in html das Aufzählungszeichen festgelegt werden:

Beispiel: ul.a { list-style-type: circle; } ul.b { list-style-type: square; }

Eigenes Aufzählungszeichen einfügen:

Beispiel:
ul {
list-style-image: url('sqpurple.gif');
}

Position des Aufzählungszeichen wählen

Mithile des Kommandos list-style-position legt man entweder insinde oder outside fest.

Beispiel:
ul.a {
list-style-position: outside;
}
ul.b {
list-style-position: inside;
}

Um die Aufzählungszeichen auszuschalten, also keine anzuzeigen, benutzt man den Kommand list-style-type:none

Um die Einrückungen zu entfernen, muss man "margin" und "padding" auf 0 setzen.

Beispiel:
ul {
list-style-type: none;
margin: 0;
padding: 0;
}

HTML Tables

Nun werden die Grundkommandos bzw Tags für die Erstellung eines Tables in HTML aufgezählt:

  1. < table > Der Tag, um einen neuen Table zu erstellen
  2. < tr > Deffiniert eine neue Zeile (Row)
  3. < th > Die Header werden mit einem th eingeleitet bzw deffiniert
  4. < td > Steht für table data und wird für die Deffinierung neuen Contents verwendet
Beispiel für einen Table:
Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

Borders

Um eine Border zu zeichnen, verwendet man den command border:

Beispiel: table,th,td{
border: 1px solid black;
}

Nun kann es natürlich sein, dass man nur eine Umrandung (Linie) hat. Dafür schreibt man unter dem border: noch border-collapse: collap

Beispiel: table,th,td{
border: 1px solid black; border-collapse: collapse; }

Um die Größe der Zellen zu ändern verwendet man den Kommand padding Xpx

Mithilfe des Kommands text-align: right/centre/left kann man die Positionierung des Textes ändern.
